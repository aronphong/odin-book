# odin-book

Social media application built as my final project for The Odin Project's Javascript Course.

Users can also create small posts, like/dislike and comment to other posts.

Redux is used for state management.

## Features

- User authentication with JSON Web Token
- Friends Requests and Friends
- Text posts
- Likes and comments

### Features I'm Working on

- Photo posts
- Facebook authentication
- Friend requests and friends
- Photo Uploading

### Things I need to fix
